# robosys2022
ロボットシステム学2022



## plus
* 標準入力から読み込んだ数字を足す。
* seq 5 | ./plus を使って実行する。
* 実行結果
```
$ seq 5 | ./plus
15
```
  
              


## 必要なソフトウェア
* Python
  * テスト済み: 3.7~3.10
